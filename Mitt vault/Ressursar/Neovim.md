## Inserting / editing

|   |   |
|---|---|
|i|insert text|
|A|append to end of line|
|||

## Files

|vim .|directory listing of current dir|
|%|create file|

|:Ex|explore|

## Navigation

### File / go to line

|   |   |
|---|---|
|gg|start of file|
|G|end of file|
|<##>G|go to line number <##>|

### Line

|   |   |
|---|---|
|0|start of line|
|$|end of line|
|dd|delete line|

### Clipboard

|   |   |
|---|---|
|y|yank (copy)|
|p|paste|
|yy|copy line|
|||

### Search

|   |   |
|---|---|
|/needle|forward search for needle|
|n|next result|
|N|previous result|
|?needle|backward search for needle (n = next result backwards, N = previous result backwards, i.e. forwards)|
|||
