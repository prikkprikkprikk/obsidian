On iOS or iPadOS, make sure that you last played the world you want to export.

Then open the **Files** app, select **On My iPhone** or **On My iPad** and open ![File directory.png: Sprite image for directory in Minecraft](https://minecraft.wiki/images/thumb/File_directory.png/16px-File_directory.png?8a409) `Minecraft/games/com.mojang/minecraftWorlds`. 

Now set the sorting order to newest first, open the first folder and verify that it's the correct one by checking the ![File file.png: Sprite image for file in Minecraft](https://minecraft.wiki/images/thumb/File_file.png/16px-File_file.png?e19ce) `levelname` file. 

Finally select all of it's contents, compress them and rename the resulting zip archive to `_world name_.mcworld`.

You can now share this file with others.

## Alternativt
### Step 1 — Export the world from your child's iPad

1. Open Minecraft on her iPad, go to **Play**
2. Find her survival world, tap the **pencil icon** to edit it
3. Scroll down and tap **Export World**
4. Save the `.mcworld` file somewhere accessible — **iCloud Drive** is the easiest option on iPad

---

### Step 2 — Get the file to a device with your account

**Option A (same iPad, different account):**

- The `.mcworld` file is just sitting in Files/iCloud — it doesn't matter which Minecraft account is active
- Sign your child out of Minecraft and sign _yourself_ in on the same iPad
- Go to the Files app, tap the `.mcworld` file — Minecraft should offer to import it
- The world now appears in _your_ local worlds list

**Option B (different device):**

- Share the `.mcworld` file via AirDrop, iCloud Drive, or even email to a device where you're already signed into Minecraft
- Open the file from the Files app on that device to import it

---

### Step 3 — Upload to the Realm (as owner)

Now logged in as yourself with the world imported:

1. Go to **Play** → find your Realm → tap the **pencil icon**
2. Tap **Edit World** on the slot you want to use
3. Tap **Replace World**
4. Select the imported world
5. Confirm — and it's live on the Realm
